## Критические баги (БЕЗОПАСНОСТЬ)

### 1. Регистрация: XSS уязвимость
- **Тест:** `test_injection_in_name_field[xss]`
- **Проблема:** XSS payload рендерится на странице
- **Критичность:** ВЫСОКАЯ

### 2. Создание магазина: Null Byte инъекция
- **Тест:** `test_null_byte_injection`
- **Проблема:** Null bytes не санитизируются
- **Критичность:** ВЫСОКАЯ

### 3. Создание магазина: Пробелы в описании
- **Тест:** `test_description_whitespace_only`
- **Проблема:** Описания только из пробелов принимаются
- **Критичность:** СРЕДНЯЯ

## Функциональные баги

### 1. Логин: Неверные credentials
- **Тест:** `test_login_with_invalid_credentials`
- **Проблема:** Не показывается ошибка при неверных данных
- **Критичность:** СРЕДНЯЯ

### 2. Сотрудники: Создание
- **Тест:** `test_create_staff_with_valid_data`
- **Проблема:** Нет обратной связи после отправки формы
- **Критичность:** СРЕДНЯЯ

### 3. Сотрудники: Удаление
- **Тест:** `test_confirm_delete_removes_staff`
- **Проблема:** Удаление не удаляет сотрудника
- **Критичность:** ВЫСОКАЯ

### 4. Накладные: Множество ошибок
- Процесс создания сломан

- Управление количеством сломано

## Категории ERROR

### 1. Страница деталей продукта (64 ошибки)
- **Причина:** Тесты переходят на `/products/detail/{id}` без валидного ID продукта
- **Решение:** Нужна фикстура для получения валидного ID продукта

### 2. Мульти-продукт шаги (50 ошибок)
- **Причина:** Навигация на шаг 2/3 падает - состояние формы не сохраняется
- **Решение:** Необходимо заполнить шаг 1 валидными данными перед шагом 2/3

### 3. Создание продукта шаги (42 ошибки)
- **Причина:** Аналогично multiproduct - проблемы навигации wizard
- **Решение:** Обеспечить валидные данные на каждом шаге
