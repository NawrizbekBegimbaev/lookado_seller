## БАГИ БЕЗОПАСНОСТИ (Критические)

### БАГ-001: XSS уязвимость в регистрации
- **Файл:** test_registration.py
- **Тест:** `TestRegistrationSecurity::test_injection_in_name_field[xss]`
- **Описание:** XSS payload `<script>alert('xss')</script>` рендерится на странице без санитизации
- **Критичность:** КРИТИЧЕСКАЯ
- **Влияние:** Злоумышленник может внедрить вредоносные скрипты

### БАГ-002: Null Byte инъекция в создании магазина
- **Файл:** test_shopcreate.py
- **Тест:** `TestShopCreateAdvancedSecurity::test_null_byte_injection`
- **Описание:** Null bytes (`\x00`) не санитизируются в полях форм
- **Критичность:** ВЫСОКАЯ
- **Влияние:** Возможные атаки через null byte инъекцию

### БАГ-003: XSS в создании продукта

## КАТЕГОРИИ ERROR: Отсутствуют предусловия тестов

### ERR-001: Тесты деталей продукта (64 ошибки)
- **Причина:** Тесты переходят на `/products/detail/{id}` без валидного ID продукта
- **Требуется исправление:** Добавить фикстуру для получения валидного ID из списка продуктов

### ERR-002: Тесты шагов мульти-продукта (50 ошибок)
- **Причина:** Тесты шагов 2/3 падают, потому что шаг 1 не завершен
- **Требуется исправление:** Обеспечить ввод валидных данных на шаге 1 перед переходом

### ERR-003: Тесты шагов создания продукта (42 ошибки)
- **Причина:** Шаги wizard зависят от завершения предыдущего шага
- **Требуется исправление:** Правильно связать зависимости тестов
